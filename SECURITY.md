# Security Policy

## Supported Versions

We currently provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you find a security bug, please **do not** open a public issue. 

Instead, please report vulnerabilities by:
1. Emailing **elghoul.youcef01@gmail.com**
2. Providing a detailed description of the exploit
3. Including steps to reproduce the issue

I will do our best to respond within 48 hours and coordinate a fix.
